2% matRad function to return the rotation / transformation matrix 
3% for gantry and/or couch rotation. The Rotation matrix stands for a (1)
4% counter-clockwise, (2) active rotation in the patient coordinate system
5% that is performed on a (4) column vector (by premultiplying the matrix). 
6% Per change of one of these directions a matrix transpose of the returned 
7% matrix is required.

68        %patient system around the beam target point / isocenter
69        R_Couch = [cosd(couchAngle) 0 sind(couchAngle); ...
70            0 1 0; ...
