51
52% target resolution is ct.resolution

54
55% convert dosedata to 3-D cube

58% ct resolution is target resolution, now convert to new cube;

60% generating grid vectors
61x = doseInfo.ImagePositionPatient(1) + doseInfo.ImageOrientationPatient(1) * ...

81% write new parameters

91dose.dicomInfo.SliceThickness          = target_resolution.z;

93dose.dicomInfo.DoseType                = doseInfo.DoseType;
94dose.dicomInfo.DoseSummationType       = doseInfo.DoseSummationType;
