17% In this example we will show 
18% (i) how to load patient data into matRad
19% (ii) how to setup a carbon ion dose calculation plan including variable RBE optimization
20% (iii) how to inversely optimize the pencil beam intensities based on the
21% RBE-weighted dose
22% (iv) how to inversely optimize the pencil beam intensities based on the
23% biological effect
24% (v) how to change the tissues' radiobiological characteristics
25% (vi) how to recalculated the dose considering the previously optimized pencil beam intensities

28%% Patient Data Import
29% Let's begin with a clear Matlab environment and import the liver
30% patient into your workspace.
31
32matRad_rc; %If this throws an error, run it from the parent directory first to set the paths
33
34load('LIVER.mat');
35
36%% Treatment Plan
37% The next step is to define your treatment plan labeled as 'pln'. This 
38% structure requires input from the treatment planner and defines the most
39% important cornerstones of your treatment plan.
40%%
41% First of all, we need to define what kind of radiation modality we would
42% like to use. Possible values are photons, protons or carbon. In this
43% example we would like to use carbon ions for treatment planning. Next, we
44% need to define a treatment machine to correctly load the corresponding 
45% base data. matRad features generic base data in the file
46% 'carbon_Generic.mat'; consequently the machine has to be set accordingly
47pln.radiationMode = 'carbon';            
48pln.machine       = 'Generic';
49
50%%
51% Define the flavor of biological optimization for treatment planning along
52% with the quantity that should be used for optimization. Possible values 
53% are (none: physical optimization; const_RBExD: constant RBE of 1.1; 
54% LEMIV_effect: effect-based optimization; LEMIV_RBExD: optimization of 
55% RBE-weighted dose. As we use carbon ions, we decide to use base data from 
56% the local effect model IV and want to optimize the RBE-weighted dose. 
57% Therefore we set bioOptimization to LEMIV_RBExD
58pln.propOpt.bioOptimization = 'LEMIV_RBExD';                                              
59
60%%
61% The remaining plan parameters are set like in the previous example files
62pln.numOfFractions        = 30;
63pln.propStf.gantryAngles  = 315;
64pln.propStf.couchAngles   = 0;
65pln.propStf.bixelWidth    = 3;
66pln.propStf.numOfBeams    = numel(pln.propStf.gantryAngles);
67pln.propStf.isoCenter     = ones(pln.propStf.numOfBeams,1) * matRad_getIsoCenter(cst,ct,0);
68pln.propOpt.runDAO        = 0;
69pln.propOpt.runSequencing = 0;
70
71% dose calculation settings
72pln.propDoseCalc.doseGrid.resolution.x = 3; % [mm]
73pln.propDoseCalc.doseGrid.resolution.y = 3; % [mm]
74pln.propDoseCalc.doseGrid.resolution.z = 3; % [mm]
75
76%% Generate Beam Geometry STF
77stf = matRad_generateStf(ct,cst,pln);
78
79%%
80% Let's have a closer look on the stf.ray sub-structure which contains the 
81% actual  beam/ray geometry information. For illustration purposes we want 
82% to show the last ray. Besides geometrical  information about the position 
83% and orientation of the ray, we can also find pencil beam information. If 
84% the ray coincides with the target, pencil beams were defined along the 
85% ray from target entry to target exit. 
86display(stf.ray(end));
87
88%%
89% Here are the energies selected on the last ray: 
90display(stf.ray(end).energy);
91
92%% Dose Calculation
93dij = matRad_calcParticleDose(ct,stf,pln,cst);
94
95%% Inverse Optimization  for IMPT based on RBE-weighted dose
96% The goal of the fluence optimization is to find a set of bixel/spot 
97% weights which yield the best possible dose distribution according to the
98% clinical objectives and constraints underlying the radiation treatment.
99resultGUI = matRad_fluenceOptimization(dij,cst,pln);
100
101%% Plot the Resulting Dose Slice
102% Let's plot the transversal iso-center dose slice
103slice = round(pln.propStf.isoCenter(3)./ct.resolution.z);
104figure,
105imagesc(resultGUI.RBExDose (:,:,slice)),colorbar, colormap(jet);
106
107%% Inverse Optimization  for IMPT based on biological effect
108% To perform a dose optimization for carbon ions we can also use the
109% biological effect instead of the RBE-weighted dose. Therefore we have to
110% change the optimization mode and restart the optimization
111pln.propOpt.bioOptimization = 'LEMIV_effect'; 
112resultGUI_effect = matRad_fluenceOptimization(dij,cst,pln);
113
114%% Visualize differences
115% Through optimzation based on the biological effect we obtain a slightly
116% different dose distribution as visualized by the following dose
117% difference map
118figure;
119imagesc(resultGUI.RBExDose (:,:,slice)-resultGUI_effect.RBExDose(:,:,slice));
120colorbar;
121colormap(jet);
122
123%% Change Radiosensitivity
124% The previous treatment plan was optimized using an photon alpha-beta 
125% ratio of 2 for all tissues. Now, Let's change the radiosensitivity by 
126% adapting alphaX. This will change the photon alpha-beta ratio
127% from 2 to 10.
128for i = 1:size(cst,1)
129    cst{i,5}.alphaX      = 0.5;
130    cst{i,5}.TissueClass = 2;
131end
132
133%% Recalculate Plan
134% Let's use the existing optimized pencil beam weights and recalculate the RBE weighted dose
135resultGUI_tissue = matRad_calcDoseDirect(ct,stf,pln,cst,resultGUI.w);
136
137%% Result Comparison
138% Let's compare the new recalculation against the optimization result.
139plane = 3;
140doseWindow = [0 max([resultGUI_effect.RBExDose(:); resultGUI_tissue.RBExDose(:)])];
141
142figure,
143matRad_plotSliceWrapper(gca,ct,cst,1,resultGUI_effect.RBExDose,plane,slice,[],[],colorcube,[],doseWindow,[]);
144title('original plan')
145figure,
146matRad_plotSliceWrapper(gca,ct,cst,1,resultGUI_tissue.RBExDose,plane,slice,[],[],colorcube,[],doseWindow,[]);
147title('manipulated plan')
148%% 
149% At this point we would like to see the absolute difference of the original optimization and the 
150% recalculation. 
151absDiffCube = resultGUI_effect.RBExDose-resultGUI_tissue.RBExDose;
152figure,
153matRad_plotSliceWrapper(gca,ct,cst,1,absDiffCube,plane,slice,[],[],colorcube);
154title('absolute difference')
155%%
156% Plot both doses with absolute difference and gamma analysis
157[gammaCube,gammaPassRate,hfigure]=matRad_compareDose(resultGUI_effect.RBExDose, resultGUI.RBExDose, ct, cst,[1 1 1],'on');
